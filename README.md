# Celebal Excellence Data Science Internship

This repository contains assignments, notebooks, projects, and learning activities completed during the Celebal Excellence Data Science Internship Program. It serves as a structured record of my progress, covering fundamental concepts in Data Science, Machine Learning, Deep Learning, Natural Language Processing, Computer Vision, and Generative AI.

## Repository Overview

| Week    | Topics Covered                                                              |
| ------- | --------------------------------------------------------------------------- |
| Week 01 | Python Fundamentals, NumPy, Pandas, Linear Algebra, Statistics, Probability |
| Week 02 | Exploratory Data Analysis (EDA) and Machine Learning                        |
| Week 03 | Clustering Techniques                                                       |
| Week 04 | Computer Vision                                                             |
| Week 05 | Natural Language Processing (NLP)                                           |
| Week 06 | Autoencoders                                                                |
| Week 07 | Retrieval-Augmented Generation (RAG)                                        |
| Week 08 | AI Agents                                                                   |

## Learning Objectives

* Develop a strong foundation in Data Science and Analytics.
* Gain hands-on experience with Python and its Data Science ecosystem.
* Apply Machine Learning and Deep Learning techniques to practical problems.
* Explore modern AI applications, including NLP, Computer Vision, RAG, and AI Agents.
* Build a portfolio of assignments and projects demonstrating acquired skills.
